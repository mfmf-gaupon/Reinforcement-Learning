# モンテカルロ法によるCartPole

モンテカルロ法によりCartPole課題を解く
環境はOpen AI Gymから"CartPole-v1"を使用する

### 動作環境
- macOS Big Sur 11.2.3
- python 3.7.10
- gym 0.17.3


### 使用法
```zsh:
% python montecarlo.py
```
