# DQN(Deep-Q-Network)によるCartPole

強化学習用ライブラリであるTF-Agentsを使用してDQNのエージェントを作り
CartPole課題を解く  
環境はOpen AI Gymから"CartPole-v1"を使用する

### 動作環境
- Windows10
- python 3.6.10
- gym 0.17.1
- tensorflow 2.4.1
- tf-agents 0.7.1

### 使用法
```zsh:
% python tfagents_dqn.py
```
